    1. Operating Systems

    Newfiles is designed to work on both Windows and Mac computers. Have one of these? You're good to go.

    A note to Linux users: You may still be able to look at these setup instructions and figure out how to get it running. But, I've never personally tried Newfiles on Linux.

    2. Required JDK Version

    Newfiles is programmed in Java and requires a minimum Java Development Kit (JDK) version. You can download the JDK for free from the internet (of course).

    The minimum requirement is JDK 1.7.0_17.

    If you already have this, or a later version, you can skip to step 3. Otherwise, you will have to download a JDK version that will meet this minimum requirement.

    How to check your JDK version

    Open up a command prompt. In Windows, you would type cmd into the start menu, for example. This would cause the cmd.exe program to open up your command prompt window. If you don't know how to open the command prompt, you can search for directions online.

    Once the command prompt is open, type the following: java -version. This asks your computer what version of Java (if any) it has:

    As you can see my console told me I have java version "1.7.0_17". Yay! I meet the minimum requirements so I can move on to step 3 instead of having to download the required minimum version. If you aren't as lucky, just follow the next steps to get the SDK.

    How to download the minimum JDK version

    You can find additional instructions, published by Oracle, for installing the SDK:

    • Java SDK Installation instructions (Windows)
    • Java SDK Installation instructions (Mac)

    Download the JDK from Oracle's website (Oracle currently owns the Java platform, which is free to download).

    • Click here to access the download page for the Java SDK

    Should you choose the 32 or 64 bit version? Make sure you choose the one that matches the type of computer you are using. It's important to download the correct SDK version. Note: x86 is for 32-bit systems.
