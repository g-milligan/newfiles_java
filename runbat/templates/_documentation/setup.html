    2. Required JDK Version

    Newfiles is programmed in Java and requires a minimum Java Development Kit (JDK) version. You can download the JDK for free from the internet (of course).

    The minimum requirement is JDK 1.7.0_17.

    If you already have this, or a later version, you can skip to step 3. Otherwise, you will have to download a JDK version that will meet this minimum requirement.

    How to check your JDK version

    Open up a command prompt. In Windows, you would type cmd into the start menu, for example. This would cause the cmd.exe program to open up your command prompt window. If you don't know how to open the command prompt, you can search for directions online.

    Once the command prompt is open, type the following: java -version. This asks your computer what version of Java (if any) it has:

    As you can see my console told me I have java version "1.7.0_17". Yay! I meet the minimum requirements so I can move on to step 3 instead of having to download the required minimum version. If you aren't as lucky, just follow the next steps to get the SDK.

    Use Git to download Newfiles (for the pro's)

    New to Git?

    What is Git? It's an application that lets you type console commands, like git pull origin master or git push origin master in order to move files between your computer and external computers. Websites like Github.com and bitbucket.org host a backup of the files that you push to their servers. Any location that holds your files is called a repository. So... let's say you take a baseball bat to your computer. Your files are still safe on the external repository. When you get a new computer you can just git pull origin master from the external repository to get a copy of your files onto your new local machine.

    The real power of Git, however, comes from its ability to manage and merge file changes coming from more than one developer. For example, if you and your friend are developing a sweet game mod at the same time; editing the same files at the same time. When you git push origin develop Git will detect the conflict and help you decide how to merge the conflicting files. Or Git may be able to automatically merge edits from both you and your buddy.

    Right now, you don't have to know about these details... although it wouldn't hurt for you to read more about Git when you have time; knowing about Git is probably useful for you, considering you are here learning about Newfiles.
